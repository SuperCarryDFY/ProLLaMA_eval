## Running ProLLaMA

First, clone the official ProLLaMA repository:

```bash
cd src
git clone git@github.com:PKU-YuanGroup/ProLLaMA.git
```

Then run ProLLaMA inference with:

```bash
bash scripts/infer.sh
```

## Others

See `scripts/process.ipynb` for how `input.txt` is generated and how the ProLLaMA outputs are post-processed.
